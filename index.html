FAQ

Die wichtigsten Fragen vor dem Start.

Brauche ich Vorkenntnisse in funktionaler Programmierung?

Nein. Grundlagen in einer anderen Sprache reichen. Der Kurs fuehrt die Konzepte Schritt fuer Schritt ein.

Ist der Kurs eher fuer Backend oder verteilte Systeme gedacht?

Beides. Der Fokus liegt auf den Staerken von Erlang: Prozesse, Robustheit und verteilte Architektur.

Kann ich die Seite direkt auf GitHub Pages deployen?

Ja. Das Projekt ist als statische Website aufgebaut und kann ohne Build-Schritt auf GitHub Pages laufen.
